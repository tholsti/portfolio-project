Portfolio

ScratchMapp

ScratchMapp was the final project at the Coding Bootcamp. In a team of three and just five weeks of time, we completed a project where users can create a profile and save the countries of the world they've visited by clicking on an interactive map. Features also include city views showing top attractions of the city, search bar, and multiple API endpoints

Backend for the project was created with the Laravel PHP MVC framework, MySQL database and Ajax queries for asynchronous data handling. Map view was created with the MapBox Javascript platform, and further functionality as Javascript and React components.

Links: ScratchMapp demo | GitHub repository

Flight-O-Matic

The project started as a 7-hour React hackathon at the bootcamp. I've then cloned the project to create my own React playground, and written most the app fully from ground up again. Based on user input, it finds the cheapest flights between any two cities in the world within a given range of dates.
